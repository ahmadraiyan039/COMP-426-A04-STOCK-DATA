# COMP-426-A04-STOCK-DATA
Client Side Web App to see current and historical stock prices. This is a basic client side web app where user will be prompted to enter a stock ticker, and once the ticker has been entered, it will show the current price, historical price, and visualize the different pricepoints.

It uses Alpha Vantage API for real-time current price, and Financial Prep Modeling API to fetch historic pricing data, and uses cdn to graph the prices.
